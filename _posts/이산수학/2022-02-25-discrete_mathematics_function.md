#  함수



## 기본개념

- 함수는 한 집합에 있는 <strong>모든</strong> 원소가 다른집합의 원소에 <strong>한번씩</strong>만 대응되는 관계를 말한다
- 공집합이 아닌 두 집합 $$X, Y$$가 있을때 $$X$$에서 $$Y$$로의 함수 $$f$$ 는 집합$$X$$의 각각의 원소 에 집합 $$Y$$의 원소를 단 하나씩 대응시킨 것이다
- 집합 $$X$$는 정의역(domain)
- 집합 $$Y$$는 공역(codomain) 

- 집합 $$X$$에 대응되는 집합$$Y$$의 원소 $$y$$들의 집합을 치역(range) 이라고 한다



## 함수의 성질



### 단사함수

- $$a,b\in X$$ 에 대해 $$f(a) = f(b) $$이면  $$a = b$$ 일 경우 $$f$$를 단사함수(one-to-one function) 라고 한다
- 즉, $$f(a) = y$$ 이고 $$f(b) = y$$  인 $$y$$가 존재하지 않음

### 전사함수

- $$y\in Y$$ 에 대하여 $$f(x) = y$$ 인 원소 $$x\in X$$가 적어도 하나 존재할 경우 $$f$$를 전사함수(onto function)라고 한다

- 즉, 치역 = 공역

### 전단사함수

- 일대일 대응인 함수



## 합성함수



- 함수 $$f : X\rightarrow Y$$ 와  함수 $$g : Y\rightarrow Z$$에 대하여 집합 $$X$$에서 집합 $$Z$$로의 함수를 $$g\circ f : X\rightarrow Z$$로 나타내며 $$f$$와 $$g$$의 **합성함수**(composite function)라고 한다

- $$(g\circ f)(x) = g(f(x))$$
- $$h\circ (g\circ f) = (h\circ g)\circ f$$
- $$f$$와 $$g$$가 단사함수면 $$g\circ f$$ 도 단사함수
- $$f$$와 $$g$$가 전사함수면 $$g\circ f$$ 도 전사함수
- $$f$$와 $$g$$가 일대일 대응이면 $$g\circ f$$ 도 일대일 대응



## 여러가지 함수



### 항등함수

집합 $$X$$에 대한 함수 $$f :  X\rightarrow X$$가 모든 $$x\in X$$에 대하여 $$f(x) = x$$  일때 $$f$$ 를 **항등함수**(identity function)라고 하며 

$$i_x$$  로 나타낸다, 즉 $$i_x (x) = x$$

예)

 $$X = \{-1,0,1\}$$ 일때 $$f(x) = x ^3$$ 은 항등함수 인가?

$$f(-1) = -1, f(0) = 0 f(1) = 1$$ 항등함수이다



### 역함수

- $$f(x) = y$$ 일때 $$f⁻^1(y) = x 인 f⁻^1$$이 존재하면 $$f⁻^1$$ 은 $$f $$의 역함수임
- 역함수가 존재하는 경우는 함수$$ f $$가 일대일 대응일 경우이다



### 특성함수

전체집합 $$U$$ 의 임의의 부분집합$$S$$ 에 대하여 함수 $$f_s: U \rightarrow \{0,1\}$$을 다음과 같이 정의 할때 $$f_s$$를 $$S$$의 **특성함수**(characteristic)라고 한다

$$
f_s(x) = \left\{\begin{matrix}
 1 \quad x \in S 일때\\0 \quad x \notin S 일때
\end{matrix}\right.
$$

예) 전체집합$$ U = \{1,2,3,4,5,6,7\} $$일때 부분집합 $$A = \{2,3,4,5,6\}$$ 에 대한 특성함수는?

특성함수 $$ f_A $$는 다음과 같다

$$
\begin{align}
f_A(x) = \left\{\begin{matrix}
 1 \quad x =&2,3,4,5,6 일때\\ 0 \quad x  =&1,7,일때
\end{matrix}\right.
\end{align}
$$




### 최대,최소 정수함수

$$x\in R$$에 대해 $$x$$와 같거나 $$x$$ 보다 작은수 중에서 $$x$$와 가장 가까운 정수를 대응시키는 함수를 **최대정수함수**(greatest integer function)

$$x\in R$$에 대해 $$x$$와 같거나 $$x$$ 보다 큰수 중에서 $$x$$와 가장 가까운 정수를 대응시키는 함수를 **최소정수함수**(least integer function) 라고 한다
